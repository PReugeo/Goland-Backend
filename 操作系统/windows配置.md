# PowerShell 配置

Windows 为了达到 Linux zsh 效果，需要先安装 oh-my-posh，而后需要安装相关模块（zsh plugin），安装相关字体进行配置。

## 安装 oh-my-posh

我们需要先以管理员权限启动 PowerShell。

并且先安装 [Git for Windows](https://git-scm.com/downloads)，然后在 Powershell 下运行一下命令

```powershell
Install-Module posh-git -Scope CurrentUser
Install-Module oh-my-posh -Scope CurrentUser
```

> 可能会需要安装 NuGet，如果之前没有安装的话，PowerShell 会自动跳出提示安装，同意即可，如果此前没有开启执行任意脚本，此处也会提示执行脚本。如果没有权限执行脚本，可能需要先执行 `Set-ExecutionPolicy Bypass`。

Posh-Git 将 Git 状态信息以及 Git 命令，参数，远程和分支名称的制表符完成添加到提示符。 Oh-My-Posh 为PowerShell 提示提供主题功能。

[oh-my-posh 主题地址](https://github.com/JanDeDobbeleer/oh-my-posh#themes)

还可以安装类似于 `autojump` 功能的 powershell 插件

```powershell
Install-Module ZLocation -Scope CurrentUser
```

该插件使用 `z` 关键字来进行目录的跳转。

> 查看 z 的记录： `Get-ZLocation`
>
> 配置文件: `C:\Users\xxxxx\z-location.txt`

安装 PSReadLine（提供命令提示，autosuggestion 等功能）

```powershell
Install-Module -Name PSReadLine -Scope CurrentUser -Force -SkipPublisherCheck
```

然后输入一下命令行创建 Powershell Profile，如果此前没有创建过的话，来进行模块的导入和命令配置等操作

```powershell
if (!(Test-Path -Path $PROFILE )) { New-Item -Type File -Path $PROFILE -Force }
notepad $PROFILE
```

配置文件分享

```powershell
Import-Module DirColors
Import-Module posh-git
Import-Module oh-my-posh
Import-Module git-aliases -DisableNameChecking
Import-Module PSReadLine
Set-Theme robbyrussell 
Set-PSReadLineOption -PredictionSource History
# Advanced Autocompletion for arrow keys
Set-PSReadlineKeyHandler -Key UpArrow -Function HistorySearchBackward
Set-PSReadlineKeyHandler -Key DownArrow -Function HistorySearchForward

# Chocolatey profile
$ChocolateyProfile = "$env:ChocolateyInstall\helpers\chocolateyProfile.psm1"
if (Test-Path($ChocolateyProfile)) {
  Import-Module "$ChocolateyProfile"
}
```

首先切换到 UTF-8 代码页，之后导入模块 PSReadLine，并设置依据历史进行命令补全预测、以及高级的方向键历史命令搜索，PSReadLine 相关设置，可以在[PSReadLine Module - PowerShell | Microsoft Docs](https://docs.microsoft.com/en-us/powershell/module/psreadline/?view=powershell-7.1)找到

## WSL Ubuntu

在其中设置 powerLine 字体，使 powershell 支持 powerline 图标

```shell
sudo apt install golang-go
go env -w GO111MODULE=on
go env -w GOPROXY=https://goproxy.io,direct
sudo chmod 777 /home/xxx/go
go get -u github.com/justjanne/powerline-go
```

自定义 Ubuntu 中的提示

在 `~/.bashrc` 或者 `~/.zshrc` 中，在其中添加

```shell
function _update_ps1() {
    PS1="$($GOPATH/bin/powerline-go -error $?)"
}
if [ "$TERM" != "linux" ] && [ -f "$GOPATH/bin/powerline-go" ]; then
    PROMPT_COMMAND="_update_ps1; $PROMPT_COMMAND"
fi
```
